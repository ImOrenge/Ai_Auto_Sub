# 상업적으로 이용 가능한 한글 폰트 목록

## Google Fonts (SIL Open Font License)

### 고딕체 (Sans-Serif)
1. **Noto Sans KR** ✅ (이미 설치됨)
   - Variable 폰트 지원
   - 가중치: Thin(100) ~ Black(900)
   
2. **IBM Plex Sans KR**
   - 모던한 고딕체
   - 가중치: Thin(100) ~ Bold(700)
   
3. **Black Han Sans**
   - 강렬한 흑체
   - 가중치: Regular(400)
   
4. **Do Hyeon**
   - 귀여운 느낌의 고딕체
   - 가중치: Regular(400)
   
5. **Jua**
   - 둥근 고딕체
   - 가중치: Regular(400)
   
6. **Nanum Gothic**
   - 전통적인 나눔고딕
   - 가중치: Regular(400), Bold(700), ExtraBold(800)

### 명조체 (Serif)
7. **Noto Serif KR**
   - 전통적인 명조체
   - Variable 폰트 지원
   - 가중치: ExtraLight(200) ~ Black(900)
   
8. **Nanum Myeongjo**
   - 나눔명조
   - 가중치: Regular(400), Bold(700), ExtraBold(800)

### 손글씨/장식체
9. **Gamja Flower**
   - 귀여운 손글씨체
   - 가중치: Regular(400)
   
10. **Sunflower**
    - 해바라기체 (3가지 스타일)
    - 가중치: Light(300), Medium(500), Bold(700)

11. **Gowun Batang**
    - 고운 바탕체
    - 가중치: Regular(400), Bold(700)

## 다운로드 명령어

아래 명령어로 모든 폰트를 한번에 다운로드합니다:

```powershell
# PowerShell
$fonts = @(
    @{name="IBM Plex Sans KR"; file="IBMPlexSansKR-Regular.ttf"; url="https://github.com/google/fonts/raw/main/ofl/ibmplexsanskr/IBMPlexSansKR-Regular.ttf"},
    @{name="Black Han Sans"; file="BlackHanSans-Regular.ttf"; url="https://github.com/google/fonts/raw/main/ofl/blackhansans/BlackHanSans-Regular.ttf"},
    @{name="Do Hyeon"; file="DoHyeon-Regular.ttf"; url="https://github.com/google/fonts/raw/main/ofl/dohyeon/DoHyeon-Regular.ttf"},
    @{name="Jua"; file="Jua-Regular.ttf"; url="https://github.com/google/fonts/raw/main/ofl/jua/Jua-Regular.ttf"},
    @{name="Nanum Gothic"; file="NanumGothic%5Bwght%5D.ttf"; url="https://github.com/google/fonts/raw/main/ofl/nanumgothic/NanumGothic%5Bwght%5D.ttf"},
    @{name="Noto Serif KR"; file="NotoSerifKR%5Bwght%5D.ttf"; url="https://github.com/google/fonts/raw/main/ofl/notoserifkr/NotoSerifKR%5Bwght%5D.ttf"},
    @{name="Nanum Myeongjo"; file="NanumMyeongjo%5Bwght%5D.ttf"; url="https://github.com/google/fonts/raw/main/ofl/nanummyeongjo/NanumMyeongjo%5Bwght%5D.ttf"},
    @{name="Gamja Flower"; file="GamjaFlower-Regular.ttf"; url="https://github.com/google/fonts/raw/main/ofl/gamjaflower/GamjaFlower-Regular.ttf"},
    @{name="Sunflower"; file="Sunflower-Medium.ttf"; url="https://github.com/google/fonts/raw/main/ofl/sunflower/Sunflower-Medium.ttf"},
    @{name="Gowun Batang"; file="GowunBatang-Regular.ttf"; url="https://github.com/google/fonts/raw/main/ofl/gowunbatang/GowunBatang-Regular.ttf"}
)
```
